## What is covered by which license

- **Prose, methodology descriptions, and instruction text** in this repository, including all `README.md` and `SKILL.md` files, are licensed under [Creative Commons Attribution 4.0 International (CC BY 4.0)](./LICENSE-CC-BY-4.0).
- **Code, templates, schemas, scripts, configuration, and any non-prose source files** are licensed under the [MIT License](./LICENSE).

The general rule: `.md` files containing prose or methodology are CC BY 4.0; all other files are MIT.

## Attribution (required for CC BY 4.0 material)

CC BY 4.0 requires attribution. When using or adapting prose or methodology from this repository, attribute as follows:

> *[Title of the work], by Laytheo (https://laytheo.com), licensed under CC BY 4.0.*

For example:

> *The Ontological Type System, by Laytheo (https://laytheo.com), licensed under CC BY 4.0.*

If the work has been modified, indicate that changes were made.

## Why this split

Prose carries ideas; code carries function. CC BY 4.0 protects the attribution of ideas without restricting use. MIT is the standard permissive license for code, templates, and configuration where attribution is less load-bearing. Using both, with a clear boundary, is a common pattern for repositories that mix written work and software artifacts.
