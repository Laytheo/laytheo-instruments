# laytheo-instruments

Tools for theological reading, study, and thinking. Companion repository to [laytheo.com](https://laytheo.com).

This repo is the source of truth for the instruments listed on the site. Each instrument lives in its own subdirectory with a README and the artifact itself (a prompt, a SKILL.md, an Obsidian pack, a Python script).

## Categories

- **Methods.** Approaches to organizing, reading, and thinking. The Ontological Type System lives here.
- **Packs.** Collections of markdown files to drop into an Obsidian vault.
- **Scripts.** Small Python utilities for vault maintenance and batch generation.
- **Skills.** Claude Skills tuned for specific theological work.

## Layout

```
.
├── methods/
├── packs/
├── scripts/
├── skills/
├── LICENSE
├── LICENSE-CC-BY-4.0
├── LICENSING.md
└── README.md
```

Each subdirectory under a category is a self-contained instrument. Open any folder to find a README explaining what it does and how to use it.

## Use

Take what's useful. Adapt it to your own vault, your own reading, your own work. Nothing here is precious.
